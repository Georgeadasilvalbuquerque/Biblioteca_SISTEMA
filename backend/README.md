# Caso 12 – Biblioteca
Uma biblioteca comunitária precisa organizar seu acervo de livros, revistas e
materiais de apoio. Atualmente, não há controle informatizado das entradas,
saídas e disponibilidade dos itens. Isso gera dificuldades na organização interna e
no atendimento aos usuários. O sistema deverá auxiliar no controle do acervo e na
rastreabilidade das movimentações.

# DESAFIO:

Desenvolver um sistema web ou desktop para cadastro de produtos, controle de
entradas e saídas de estoque e alerta automático quando a quantidade disponível
estiver abaixo do mínimo definido.
